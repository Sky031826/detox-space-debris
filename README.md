# detox-space-debris

Space debris orbital decay analysis using TLE data from CelesTrak.This project is in early setup. Code here computes natural orbital decay rates from TLE data — a foundational tool for my broader research on space debris.

## About

I'm building this project as an independent research effort during summer 2026. The goal is to analyze how space debris orbital decay rates vary by altitude class, using historical TLE data and the sgp4 propagator.

I'm a Class 10 student in India, age 14. The project grew out of my interest in space science — including a presentation I gave on space debris at the Erudite ISRO joined space conference at T-hub last year.

## Tools used

- Python, with libraries: sgp4, requests
- Google Colab for the working environment
- TLE data from CelesTrak
